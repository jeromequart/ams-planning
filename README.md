# AMS Croix Blanche — Planning saisonnier

Application de gestion des salariés saisonniers de l'association AMS Croix Blanche.

## Fonctionnalités
- Gestion des fiches salariés (ajout, modification, suppression)
- Planification des missions avec créneaux horaires précis
- 4 types de missions : Poste de secours, Administratif, Matériel, Nettoyage/Rangement
- Vue admin : tableau de bord complet
- Vue salarié : planning personnel, heures cumulées, prochaine mission
- Données sauvegardées localement dans le navigateur

## Stack technique
- Next.js 14
- React 18
- Stockage : localStorage (navigateur)

## Développement local
```bash
npm install
npm run dev
```
Ouvrir http://localhost:3000
